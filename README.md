# 🚧 Road Accident Analysis Dashboard (Power BI)

This project focuses on analyzing road accident data to understand patterns and identify key factors that lead to accidents. The dashboard presents complex data in a simple and interactive way to support better decision-making.

---

## 🔧 Tools Used
- Power BI  
- Excel / CSV Dataset  
- DAX  

---

## 📌 Project Overview
In this project, I cleaned and analyzed accident data and built an interactive dashboard to explore:

- Total accidents and casualties  
- Fatal, serious, and slight casualties  
- Monthly trends (current year vs previous year)  
- Accidents based on vehicle type  
- Road type and light condition analysis  
- Urban vs rural accident distribution  

---

## 📈 Key Insights
- Most accidents involve cars compared to other vehicles  
- Higher number of casualties on single carriageway roads  
- Majority of accidents occur in urban areas  
- Daytime accidents are more frequent than night  
- Noticeable drop in accidents compared to the previous year  

---

## 🎯 Dashboard Features
- Interactive filters for road surface and weather conditions  
- Clear KPI cards for quick understanding  
- Visuals like bar charts, pie charts, and trend lines  
- Simple and user-friendly layout
